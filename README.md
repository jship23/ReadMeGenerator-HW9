  ## Installation
  To install necessary dependencies, run the following command:
  ```
  npm install and then node index.js
  ```
  ## Usage
  1. Run node index.js 
  2. Answer the question prompts
  3. The readme file will be generated.
  
  ## Tests
  To run tests, run the following command:
  ```
  npm test
  ```
